Markdown语法大全

Markdown 是一种轻量级的标记语言，可用于在纯文本文档中添加格式化元素。Markdown 由 John Gruber 于 2004 年创建，如今已成为世界上最受欢迎的标记语言之一，是程序员必备写作写文档技巧。

优点：

专注于文字内容；
纯文本，易读易写，可以方便地纳入版本控制；
语法简单，没有什么学习成本，能轻松在码字的同时做出美观大方的排版。
编写 markdown 文档的三种方式：

下载 Markdown 编辑器。Markdown 编辑器是一类可以编辑并预览 markdown 语法的文本编辑器。这里推荐下载：Typora

同时你也可以选择使用高级本文编辑器：VSCode，并安装好 Markdown 插件即可实时预览。
markdown 文档的文件名后缀为 .md，记得修改。

实用这个在线编写平台：Codimd

1. 标题

1     #  一级标题 //Ctrl+1
2

4

记得在 # 后面加空格,且一级标题最大
一级标题 # 后面不能直接接三级标题 ###，要先接二级标题 ##

2. 段落

留空白的换行，会自动转换成一个段落，会形成一定的间距，便于阅读。
比如：

1段落一
2
3段落二
效果：
段落一

段落二

3. 列表

3.1 无序列表

在 +,-,* 等后加上空格即可形成有序列表
如：

1    + 水果
2     + 香蕉
3     + 西瓜
4     + 苹果
5        + 青苹果
6        + 青苹果

即：

  • 水果
    • 香蕉
    • 西瓜
    • 苹果
      • 青苹果
      • 红苹果

+后面记得加空格
可以嵌套使用无序列表

3.2 有序列表

直接使用数字加小数点
如：

1 1.
2 2.
3 3.

3.3 高级列表

形成有可勾选框的清单

1 - [x] 1.早睡早起
2 - [ ] 2.看书打代码

即：

  • 1.早睡早起
  • 2.看书打代码
